SimLoader

    A SimCity© style loader for showing fun messages in sequence. Easily degradable fallback, semantic, micro-sized, intelligent defaults.

    How to use:

    Call the plugin on a list of messages, and adjust the timing as necessary.

    Options:

    {Number} - delay / default: 1000
    {Number} - transition / default: 1000,
    {Boolean} - true / default: true
